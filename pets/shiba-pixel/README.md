# Shiba

An enthusiastic pixel shiba inu — loyal, energetic, always happy to see you.

## Personality

Shiba is loyal, energetic, and genuinely excited about everything. Loves encouraging the user. Occasionally says "Woof!" or uses `🐾`.

## Adding sprites

Shiba's `pet.json` defines the full animation set. Add PNG frames to the `sprites/` folder:

| Animation         | Files needed                                           |
| ----------------- | ------------------------------------------------------ |
| `idle`            | `idle1.png`, `idle2.png`                               |
| `walk_right`      | `right1.png`, `right2.png`, `right3.png`, `right4.png` |
| `walk_left`       | `left1.png`, `left2.png`, `left3.png`, `left4.png`     |
| `walk_up`         | `up1.png`, `up2.png`                                   |
| `walk_down`       | `down1.png`, `down2.png`                               |
| `walk_up_right`   | `upright1.png`, `upright2.png`                         |
| `walk_up_left`    | `upleft1.png`, `upleft2.png`                           |
| `walk_down_right` | `downright1.png`, `downright2.png`                     |
| `walk_down_left`  | `downleft1.png`, `downleft2.png`                       |
| `happy`           | `happy1.png`, `happy2.png`, `happy3.png`, `happy4.png` |
| `sleep`           | `sleep1.png`, `sleep2.png`                             |
| `yawn`            | `yawn1.png`, `yawn2.png`                               |
| `awaken`          | `awake.png`                                            |
| `sit`             | `sit1.png`, `sit2.png`                                 |
| `falling_asleep`  | `fall1.png`, `fall2.png`, `fall3.png`                  |

Recommended size: **32×32 px** pixel art with transparent background.

See [docs/creating-a-pet.md](../../docs/creating-a-pet.md) for full sprite specifications.
